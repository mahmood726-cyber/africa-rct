Methods

Data source: ClinicalTrials.gov API v2 (queried March 2026). All studies with at least one site in an African country were included. Comparator datasets: US trials (159,196), China (38,933), Brazil, India, and 25 Global South nations.

Novel metrics: The Clinical Contrast Index (CCI) measures the ratio of disease burden share to trial share, normalized per capita. A CCI of 1.0 indicates proportional representation; values above 1.0 indicate under-research relative to burden. The Extraction Index and Decolonization Score are composites described in their respective project dashboards.

Statistical methods: Descriptive statistics, chi-squared tests, negative binomial regression, Kaplan-Meier time-to-completion, bootstrap confidence intervals (10,000 replicates), Monte Carlo simulation, interrupted time series (ITS), difference-in-differences (DiD), synthetic control, and network centrality analysis. All code is Python 3.11+ with standard scientific stack (pandas, scipy, statsmodels, networkx).

Limitations: ClinicalTrials.gov under-represents trials registered on regional platforms (PACTR, WHO ICTRP). Burden data from GBD 2021. Population estimates from UN World Population Prospects 2024 revision. CCI calculations use disease-specific burden denominators which may not capture comorbidity overlap.
