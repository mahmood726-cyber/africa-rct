# Global Clinical Research Equity Repository

A comprehensive, multi-dimensional audit of the global clinical research landscape, analyzing 57 dimensions of inequity across Africa, Europe, China, India, and South America.

## Project Structure

- **/E156**: 57 Micro-papers (exactly 156 words each) and Interactive Dashboards.
- **/scripts**: Reproducible Python analysis scripts using ClinicalTrials.gov API v2.
- **/data**: Cached JSON metadata and statistical results.
- **PROTOCOL.md**: Formal methodology for the 57-part audit.

## The 57 Dimensions of Inequity

1.  **Core Volume & Equity:** Total counts, deep protocol metrics, structural gaps.
2.  **Advanced Physics & Math:** Escape velocity, fluid dynamics, network entropy.
3.  **Forensic & Evolutionary:** Zombie trials, survival fitness, semantic echoes.
4.  **Sovereignty & Governance:** Data ownership, sharing mandates, IPD sharing.
5.  **Global Hubs:** China Surge, India Paradox, South American Isolation.
6.  **Methodological Rigor:** Randomization, masking depth, endpoint density.
